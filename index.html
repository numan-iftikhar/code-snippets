CSS

Center a div with flexbox 
          
div{
  display: flex;
  justify-content: center;
  align-items: center;
  height: 500px; /* use desired height */
}
Center a div with grid 
            
div{
  display: grid;
  place-items: center;
  height: 500px; /* use desired height */
}

JavaScript

Reverse a string using array 
            
function reverseStr(str){
  return (str.split('').reverse().join(''));
}
